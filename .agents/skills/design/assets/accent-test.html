Substrate test only — dark mode is a departure from the skill's white-page baseline. Inks shift to their dark-substrate hex (shown second in each column header).

One process ink, doing real work.

Iteration two. The first pass treated colour as a single-mark accent (one bar, one line). The reference set instead points toward a printer's-ink data layer — a Tufte scatter where every dot is red, a planning drawing where translucent blue zones layer behind black hairlines, a Risograph page where red brackets annotate cyan halftone photography. The ink is the data and diagram surface, not a highlight.

Top: same probe set on both sides — single-ink mode, red or blue per artifact. Below the columns: Mode 3 — dual ink, red leading, blue supporting. Hue picks are printerly rather than modern UI saturated.

Earned uses of the ink
  • Whole-cloud / whole-field data marks (scatter, attractor)
  • Isolines / contours at full hairline density
  • Hatch fills carrying stratigraphic regions
  • Annotation brackets, callouts, dotted boxes over a base figure
  • Hatched fill of the highlighted bar / one ink-coloured value
Still forbidden
  • Links, buttons, body prose
  • Section rules, dividers, chrome paint
  • Backgrounds, header stripes
  • Decoration with no data
